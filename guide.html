자리배치 PWA 사용법

책상 배열 → 학생 입력 → (선택사항: 성별/고정좌석/세부 옵션 등) → 자동 배치

← 앱으로 돌아가기

빠른 시작

  1. 책상 배열에서 교실 책상 형태/행·열을 선택해요.
  2. 학생 입력에서 이름을 입력하고 저장해요. (빈 행은 자동 제외)
  3. 필요하면 (선택사항)으로 아래 기능을 설정해요:
    • 성별 지정: 성별에 따라 자리 배치
    • 고정 좌석: 특정 학생을 특정 자리로 고정
    • 세부 옵션: 금지쌍/인접 판단/모둠 관련 옵션 등
  4. 자동 배치를 눌러 좌석을 배치해요.
  5. 배치가 마음에 들지 않으면 메인창에서 책상 이동/삭제로 미세 조정할 수 있어요.

저장/불러오기 · 공유

  • 새로 저장: 현재 배치도를 새 이름으로 저장
  • 저장(덮어쓰기): 선택한 배치도를 현재 상태로 덮어쓰기
  • 불러오기/삭제: 목록에서 배치도를 선택 후 실행
  • 공유: 현재 배치도를 다른 PC/모바일로 옮길 수 있는 링크 생성 (링크는 생성 시점의 상태예요)

인쇄/이미지 다운로드

  • 이미지 다운로드(PNG)와 인쇄는 수업 자료로 활용하기 좋아요.
